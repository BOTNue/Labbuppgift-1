# Lab Assignment 1
## Description
This is a monopoly game programmed in C.
This game is not a full replica as many 
Functions are left out. the program allows
the user to choose how many players there 
are with a max of 4. The Player can walk 
around the board and buy property's using
the players money. This is done by using
switch, pointers, if statements and while 
loops.
## Installation
Install the program by cloning this
github repository.
## User Instructions
Running the program will note out what
inputs must be entered, for the user the
user to do something.
